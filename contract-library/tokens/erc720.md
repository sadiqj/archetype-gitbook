# ERC721

{% code title="erc720.arl" %}
```ocaml
archetype erc721

constant name : string = "name"

constant symbol : string = "my_token"

asset token {
  id : string;
  owner[%delegable%] : role;
}

action transfer_ (k : role, tk : pkey of token) {
  called by token[tk].owner
  effect {
    token.update(tk, {owner = k} )
  }
}

```
